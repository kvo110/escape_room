# Escape Room Challenge

Static HTML/CSS-only microsite for CSC 4370/6370 Project 1.

Pages:

- index.html (home)
- rooms.html (room cards + CSS-only reveals)
- clues.html (clue board using :target)
- final.html (CSS-only unlock moment)
- team.html (time tracking + development notes)

How to run:
Open index.html in a browser, or use a simple local server.
