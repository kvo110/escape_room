# Escape Room Challenge

CSC 4370 – Web Programming Project 1

Author:
Kenny Vo

Project Description:
This project is a CSS-driven escape room website built using semantic HTML and responsive CSS.
The experience simulates navigating a laboratory evacuation scenario where users explore puzzle rooms, gather clues, and unlock the final exit.

Technologies Used

- HTML5
- CSS Grid
- CSS Flexbox
- CSS animations
- CSS transitions
- CSS transforms

Key Features

- Responsive layout
- CSS-only clue reveal system
- interactive unlock console
- animated evacuation route map
- cinematic neon laboratory theme

Project Structure
index.html – homepage
rooms.html – puzzle rooms
clues.html – clue board
final.html – final unlock challenge
team.html – project documentation
styles.css – styling and animations
